## 1] Project Title / Headline
### Blinkit E-Commerce Sales Performance Dashboard
An interactive data analysis and visualization project built using SQL and Power BI to analyze sales trends, product performance, outlet efficiency, and revenue distribution across categories.

## 2] Short Description / Purpose
The Blinkit Sales Dashboard is a dynamic Power BI report designed to analyze transactional sales data and uncover insights related to revenue trends, product categories, outlet performance, and customer purchasing behavior.

This dashboard helps business stakeholders, retail managers, and data analysts make data-driven decisions to improve sales performance and operational efficiency.

## 3] Tech Stack
The dashboard was built using the following tools and technologies:
 Power BI Desktop – Data visualization and dashboard development
 DAX (Data Analysis Expressions) – Calculated measures and KPI metrics
 SQL – Data validation, KPI cross-checking, and business query analysis
 Power Query – Data transformation and preparation
 Data Modeling – Table relationships for cross-filtering and aggregation
 File Format – .pbix for development and .sql for query validation

## 4] Features / Highlights
###Goal of the Dashboard

To develop an interactive business intelligence tool that:
Enables sales performance monitoring
Identifies high and low-performing categories
Evaluates outlet contribution to total revenue
Supports data-driven decision making
Validates KPI accuracy using SQL queries

###Walkthrough of Key Visuals
Key KPIs (Top Section)
Total Sales
Total Items Sold
Average Rating
Average Sales per Item
These KPIs provide an immediate snapshot of business performance.

-Sales by Item Type (Bar Chart) - 
Displays revenue distribution across product categories to identify top-performing segments.

-Sales by Outlet Location Type (Donut Chart) - 
Highlights revenue contribution from Tier 1, Tier 2, and Tier 3 cities

-Outlet Establishment Trend (Line Chart) - 
Shows growth of outlets over time to analyze expansion trends.

-Fat Content Analysis (Bar/Donut Chart) - 
Compares Low Fat vs Regular product sales contribution.

-Outlet Size vs Sales (Stacked Chart) - 
Analyzes how outlet size impacts total revenue generation.

### SQL Validation

SQL queries were written to:

Verify total sales calculations
Cross-check aggregated revenue values
Validate category-level grouping
Ensure KPI consistency between database and dashboard
This improved data integrity and reporting accuracy.

## 5] Business Impact & Insights
Identified top-performing product categories driving majority revenue

Revealed outlet size and location type contributing highest sales share

Helped evaluate sales distribution across urban tiers

Improved KPI accuracy through SQL-based validation

Enabled structured reporting and performance benchmarking
